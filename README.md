# Wonder-UI 组件库

## 一套适合开发者使用的轻量级UI组件库

***
### 下载安装
```
npm install wonder-ui

cnpm install wonder-ui

yarn add wonder-ui
```
### 引入使用
```
import React from 'react';
import { Space, Button } from 'wonder-ui';

export default function Demo() {
  return (
    <Space>
      <Button type="primary" round>圆角按钮</Button>
    </Space>
  )
}
```
  #### 项目启动

  ```javascript
  yarn dev
  ```

  #### 项目打包

  ```javascript
  yarn build
  ```
